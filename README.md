## Conventions

[FE Code Convention](https://github.com/DEV-TINO/Festino/wiki/FE-Code-Conventions)

## Project Setup

> Using node lts version is recommended

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
